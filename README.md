## Run Locally
```bash
git clone https://github.com/gosiast/NextJs-portfolio.git
cd NextJs-portfolio

# Install dependencies
npm install
# or
yarn install

# Run development server
npm run dev
# or
yarn dev
```

[Open]([http://localhost:3000](https://malgorzata-stano.netlify.app/)) to view it.
